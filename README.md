# Todo List

A Simple To-do List which is built on Vite+React.

## Page
- [Todo List](https://cykrome007.github.io/Todo)
## Features

- Light/dark mode toggle
- Generates Numbers and Symbols
- Set Length of password max 20
- Copy password

- CSS
- Vite+React
- JavaScript
